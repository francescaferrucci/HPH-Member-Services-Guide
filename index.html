‚úÖ Check-In Protocol

Basic Formula

Appt type, time: reason for visit - presenting complaint
Example: MA, 10:00: Recheck liver values

Key Rules:

  • Reason for visit is what category was selected while booking
  • Time is always entered in military format
  • Pet's labeled time should be appointment time, NOT check-in time

Check-In Types

Drop-Offs (AM/PM)

Format: D/O AM/PM: reason - presenting complaint
Example: D/O AM: wellness - acupuncture
Action: Admit into 'Ready for Exam'

Walk-Ins

Format: W/I [time]: reason - presenting complaint
Example: W/I 14:05: injury - torn toenail
Action: Admit into 'Ready for Exam', highlight (Pet Parent Waiting)

Medical Appointments

Format: Medical [time]: reason - presenting complaint
Example: Medical 09:00: wellness - annual PE/Vx/TNT
Action: Leave in outpatient, PAGE BACK to notify
Late Arrivals: If 10+ minutes late, ask nurse if can still be seen. If not, nurse will advise drop-off or walk-in.
